# Sistema cafeteria
## Descrição:
Sistema da cafeteria "O Prego"

## Requisitos funcionais
- Cadastrar item no cardápio
- Alterar item no cardápio
- Deletar item do cardápio
- Buscar item do cardápio
- Listar itens do cardápio
